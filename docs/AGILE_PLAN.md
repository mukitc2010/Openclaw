# AGILE_PLAN

## Epics
- EP-1 Project Intake and PM Intelligence
- EP-2 Agile Planning and Agent Assignment
- EP-3 Technical Planning and GitHub Delivery

## Features
- Intake and project metadata capture
- PM scope and roadmap generation
- Story and task generation
- Sprint planning and assignment matrix
- Deliverable generation

## Sprint Plan
- Sprint 0: Foundation setup
- Sprint 1: Core planning flow
- Sprint 2: Hardening and auth integration
- Sprint 3: Release readiness

## Definition of Ready
- Project summary and goals provided
- Constraints and priorities captured
- Acceptance criteria draft generated

## Definition of Done
- All Sprint 1 stories have generated outputs
- Task board shows status and dependencies
- Required docs are generated and visible
