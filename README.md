# Openclaw

OpenClaw is an enterprise-grade autonomous AI Software Engineering Organization platform.
It orchestrates specialized AI agents through Agile methodology and SDLC best practices to transform a project idea into a professionally planned, architected, engineered, tested, documented, and delivered software system.

## Deployment

OpenClaw is designed to run on Amazon EC2 using Docker.

👉 **[EC2 Deployment Guide](EC2_DEPLOYMENT.md)** — step-by-step instructions for setting up OpenClaw on your EC2 instance.

### Quick Start (EC2)

```bash
curl -fsSL https://raw.githubusercontent.com/mukitc2010/Openclaw/main/devops/ec2-setup.sh -o ec2-setup.sh
chmod +x ec2-setup.sh
./ec2-setup.sh
```

See [EC2_DEPLOYMENT.md](EC2_DEPLOYMENT.md) for full configuration and troubleshooting details.
